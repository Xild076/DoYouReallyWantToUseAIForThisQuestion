DYRWTUAFTQ

DO YOU REALLY WANT TO USE AI FOR THIS QUESTION?

Intercept prompts to save computation and block slop.
Demand explicit confirmation on borderline queries.
Force Google searches into 'Web' mode (udm=14) to disable AI-generated summaries.
SETTINGS SAVED.
Please note that this extension is a personal project run on a free hosting service (HF Spaces) and trained on a small and limited dataset. It will produce false positives, false negatives, and sometimes might not even connect to the backend. Please use this more as a reminder to think critically about whether you really should be using AI :D.
