# Do You Really Want to Use AI For This Question?

Using ai to make ai unemployed!!!

Basically, the premise is: do you really need to use ai for this question? I built a mini model off of a really small dataset that checks pretty simply if the prompt you sent was a task, a high-complexity question (that would need ai), or a low-complexity question (that would not need ai).
